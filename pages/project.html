프로젝트 개요

본 프로젝트는 웹 크롤링을 통해 구글 검색 기록 데이터를 수집하고, 사용자가 입력한 키워드의 빈도를 분석하여 개인화된 검색 결과를 제공하는 시스템입니다.

적용된 알고리즘

그래프 이론

웹 페이지와 하이퍼링크를 노드와 에지로 표현하여 데이터 간의 관계를 분석하고, 효율적인 웹 크롤링을 수행합니다.

적용 분야:

  • 웹 크롤링 경로 최적화
  • 데이터 관계 분석
  • 네트워크 구조 모델링

불리언 검색

AND, OR, NOT 연산자를 사용하여 검색 쿼리를 정교하게 파싱하고, 관련성이 높은 결과를 제공합니다.

적용 분야:

  • 검색 쿼리 파싱
  • 키워드 필터링
  • 결과 정확도 향상

자연어 처리

텍스트 분석을 통해 문장의 의미를 파악하고, 특정 키워드가 포함된 문장을 추출합니다.

적용 분야:

  • 텍스트 의미 분석
  • 키워드 추출
  • 문장 분류

휴리스틱 탐색

키워드 빈도를 추적하여 우선순위를 설정하고, 검색 결과의 효율성을 높입니다.

적용 분야:

  • 검색 우선순위 결정
  • 효율적 데이터 탐색
  • 성능 최적화

문자열 매칭

수집된 텍스트에서 사용자가 입력한 키워드가 포함된 문장을 효율적으로 찾아냅니다.

적용 분야:

  • 키워드 검색
  • 패턴 매칭
  • 문장 필터링

정렬 알고리즘

필터링된 단어들을 정렬하여 사용자가 보기 쉽게 결과를 제공합니다.

적용 분야:

  • 결과 정렬
  • 데이터 구조화
  • 사용자 경험 개선

구현 세부사항

1. 파일 입출력 기능

사용자가 입력한 키워드를 파일에 저장하고, 프로그램 실행 시 이전에 입력된 키워드를 불러와 지속적인 학습을 지원합니다.

2. 웹 크롤링 기능

사용자가 입력한 키워드를 기반으로 구글 검색 결과에서 추천 검색어와 해당 키워드가 포함된 문장을 자동으로 수집합니다.

3. 키워드 빈도 분석

사용자 입력 키워드의 빈도를 추적하고, 특정 키워드가 일정 횟수 이상 입력되면 우선순위로 설정하여 개인화를 구현합니다.

4. 결과 출력 및 필터링

수집된 문장에서 단어를 추출하고, 특정 키워드가 포함된 단어만을 필터링하여 관련성 높은 결과만 출력합니다.
